---
title: "About `pip-licenses`"
date: "2020-10-08T14:56:38+01:00"
tags: ["Python"]
---

Which is a useful Python library that lists all the licenses of packages
installed with `pip`:

```sh
(venv) $ pip-licenses
 Name    Version  License
 Django  2.0.2    BSD
 pytz    2017.3   MIT
 ...
```

See <https://github.com/raimon49/pip-licenses>
