On September 25th, 2020, I learnt ...

The MacOS keybinds for en- and em-dashes

Which are:

  • alt + - for an en-dash (–);
  • shift + alt + - for an em-dash (—);

You can always get them from the ctrl + cmd + space character picker but good to know quicker alternatives.

Don’t seem to work in MacVim though.
