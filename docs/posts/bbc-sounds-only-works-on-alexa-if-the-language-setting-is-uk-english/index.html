On January 7, 2021, I learnt ...

BBC Sounds only works on Alexa if the language setting is UK English

Was puzzled as to why:

Alexa, play BBC radio four

wasn’t triggering the BBC Sounds skill on a new 4th Generation Echo.

Turns out the problem was the default language was “English (US)” when it should have been “English (UK)”.

This is noted in the BBC docs.
