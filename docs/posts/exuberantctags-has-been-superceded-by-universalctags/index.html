Exuberant-ctags has been superceded by Universal-ctags

In the sense that the Exuberant ctags project has stalled, and Universal ctags has been introduced to continue development.

I discovered this while investigating why the --python-kinds flag didn’t seem to work with Exuberant ctags.

On MacOS, install Universal ctags with:

brew install --HEAD universal-ctags/universal-ctags/universal-ctags
