On August 11th, 2020, I learnt ...

You can have SASS asset pipelines in Hugo

Which lets you build a CSS stylesheet via something like:

{{ $style := resources.Get "css/main.scss" | resources.ToCSS }}
<link rel="stylesheet" href="{{ $style.Permalink }}">

This is rather nice as you get the usual live reloading functionality when using hugo server. In previous Hugo projects, I had a separate process for rebuilding CSS, which can now be factored away.

This feature is part of Hugo’s asset pipeline functionality and has been available since v0.43.

See https://gohugo.io/hugo-pipes/scss-sass/
