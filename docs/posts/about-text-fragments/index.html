About text fragments

Modern versions of Chrome and Edge support Text Fragments, which let you specify a text snippet in the URL fragment that the browser can highlight (or bring to the user’s attention in another way).

For example, opening https://wicg.github.io/scroll-to-text-fragment/#abstract:~:text=user%20agent%20can%20quickly%20emphasise in Chrome will open the page with “user agent can quickly emphasise” highlighted:

Note the :~:text=$query in the URL.

There’s a Google-authored Chrome plugin that eases the process of building one of these URLs.
