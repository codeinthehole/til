About Vim’s :helpgrep command

Which lets you search Vim’s help files for a given pattern. For example:

:helpgrep What is the meaning of life

Discovered via a Hashrocket TIL and an episode of Josh Branchaud’s Vim Un-Alphabet screencast series.
