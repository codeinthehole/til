About the difference between minimising and hiding windows in MacOS

Minimising:

  • ⌘ + m minimises the active application window.

  • Window icon appears in Dock.

  • Selecting application via the ⌘ + tab application switcher doesn’t restore window.

Hiding:

  • ⌘ + h hides all windows of active application.

  • ⌘ + ⌥ + h hides all windows of all applications except the active one (useful).

  • Window icon not show in Dock.

  • Selecting application via the ⌘ + tab application switcher restores window.

A bit more info here: https://apple.stackexchange.com/questions/9048/what-are-the-differences-between-minimizing-and-hiding-an-application/9049
