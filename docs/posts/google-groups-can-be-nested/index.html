On January 31, 2022, I learnt ...

Google Groups can be nested

You can do this from the “Add members” section of the Group detail page. See the official docs.

This provides a useful mechanism for organising groups once there’s a large number of members.

Other things learnt about Google Workspace:

  • Email Log Search in Google Workspace has some strange behaviour
  • GSuite’s “Groups Admin” role can read any group email
  • You can hide night-time hours in Google calendar
