About pip-licenses

Which is a useful Python library that lists all the licenses of packages installed with pip:

(venv) $ pip-licenses
 Name    Version  License
 Django  2.0.2    BSD
 pytz    2017.3   MIT
 ...

See https://github.com/raimon49/pip-licenses
