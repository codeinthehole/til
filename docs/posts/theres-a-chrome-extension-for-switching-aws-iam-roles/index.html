There’s a Chrome extension for switching AWS IAM roles

It’s the AWS Extend Switch Roles extension.

AWS’s Console only keeps your 5 most recent AWS accounts as quick links, which this extension gives doesn’t limit you.

This is useful if you’re using AWS Organizations and have lots of child AWS accounts that you need to assume roles into.
