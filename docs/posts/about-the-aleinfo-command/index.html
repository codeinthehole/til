About the :ALEInfo command

The ALE plugin for Vim includes a :ALEInfo command that shows how ALE is configured for the current file and the output of any commands that have run. This is useful for debugging if your linters or fixers aren’t behaving. https://github.com/dense-analysis/ale#5ii-how-can-i-see-what-ale-has-configured-for-the-current-file
