How to scroll horizontally with a mouse

If you hold shift while using the mouse wheel then you can scroll horizontally. This is useful for wide programs like Asana and Trello.

Source: https://osxdaily.com/2012/01/26/scroll-horizontally-with-a-mouse-wheel-by-holding-shift/
