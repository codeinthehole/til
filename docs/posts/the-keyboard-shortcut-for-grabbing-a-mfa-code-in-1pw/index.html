The keyboard shortcut for grabbing a MFA code in 1PW

After opening the 1Password mini window (with cmd + option + \), select the item you want and hit shift + ctrl + cmd + c to copy the MFA code to your clipboard.

This compliments shift + cmd + c, which copies the password.
