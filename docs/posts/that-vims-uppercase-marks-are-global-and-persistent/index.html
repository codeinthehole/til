On May 6, 2021, I learnt ...

That Vim’s uppercase marks are global and persistent

Basic stuff but I didn’t realise that setting an uppercase mark (e.g. mT) creates a global bookmark that persists between sessions. I need to use this more.

See https://vim.fandom.com/wiki/Using_marks
